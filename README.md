# oracaus.dev

The apex site for the Oracaus house brand: a hub, writing on **real-time UI
performance**, work, and about. Built with [Astro](https://astro.build),
statically rendered, deployed to GitHub Pages.

```bash
npm install
npm run dev
```

Sibling properties: the library `@oracaus/coherent-derivation` and the live
demo at [demo.oracaus.dev](https://demo.oracaus.dev) live in the `oracaus`
monorepo and deploy separately.

By Przemyslaw Kalka, front-end systems engineer working on real-time risk and
trading interfaces across FX, fixed income, derivatives, and commodities.
